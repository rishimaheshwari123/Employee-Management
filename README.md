## Overview  </br>
1. Create a login script using NodeJS. Display "Login Success" on successful login; otherwise, display "Login Failed." Use MongoDB.
2. Create a NodeJS application to Insert data into the Employee Collection and populate the Employee Name Drop-down list. Upon selection of a particular Name, provide delete and view functionality.
